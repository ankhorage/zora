# Photo Social

Real Expo Router + React Native Web example app for ZORA.

This app proves an Instagram-style `social_community` starter shape with visual social routes, static image content, and real ZORA navigation chrome.

## Routes

- Home
- Explore
- Create
- Activity
- Profile

## Run

```bash
bun install
bunx expo start
```

## Rules

- Uses real Expo Router route files.
- Uses public `@ankhorage/zora` exports for UI.
- Does not use `StyleSheet`.
- Does not import `@ankhorage/surface`.
- Installs the published `@ankhorage/zora` package.
